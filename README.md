### How to start a project locally?
#### mvn clean
#### mvn compile
#### mvn package
#### java -jar target/app-1.0.0.jar


#### if you do not have mvn or jvm on your computer, you can use docker:
##### docker build -t my-application .
##### docker run -d -p 8080:8080 my-application
